## [v1.0.0] – Initial static website release

### Added
* Fully responsive static website built with HTML, CSS, and JavaScript
* Header and sticky navigation bar with smooth scrolling
* Burger menu for mobile navigation
* Accordion-style project sections with expandable details
* Interactive contact form with client-side validation and success message
* Clean semantic HTML and modern CSS styling with variables
* Minimal, vanilla JavaScript for interactivity
* Organized project structure: `index.html`, `style.css`, `script.js`
